# turing-pattern-additive-synth

## Project Background
This project is kind of a branch of my (Garrett Compston) senior capstone in Northeastern University's music technology program. The project involved
the sonification of [Turing Patterns](https://en.wikipedia.org/wiki/Turing_pattern) being used to model the development of limbs. If you are interested in the full project,
you can read my paper on the process found in this repo. Here's a [link](https://github.com/gartastrophe/turing-pattern-additive-synth/blob/main/Sonification_of_Turing_Patterns_Modeling_Limb_Development.pdf)
for quick access. There is a lot of project-specific depth in the Max patches created for all of this, but in the process, we realized that we had created a rather interesting additive
synthesizer where the timbre changes over a specified time. In fact, this synthesizer was used as the basis for a composition created to pair with an animation as a celebration of this collaboration (TODO: add link).
I wanted to release a collection of the patches created for all of this that can hopefully be used in a somewhat more standalone context. 
These files will include use of the synthesizer on its own, as well as image conversion for the sake of simulation animation which might just be of interest for synchronizing animation 
with Max using jitter. You can find more detailed instructions on using all of this further down.
